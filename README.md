<h1>Blockchain Based Online 3D Game</h1>
<p>A blockchain-based online 3D game developed using Unity and the Ropsten test network as an undergraduate graduation project.<br>
This MMORPG allows players to endlessly level up, query inventory and items on the blockchain, and trade items and ether among themselves.</p>
